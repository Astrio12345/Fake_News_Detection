# Fake_News_Detection

A Flask-based web application that uses pre-trained AI models from Hugging Face to detect fake news and analyze article credibility.

## Features

- AI-Powered Detection - Uses pre-trained RoBERTa/BERT models from Hugging Face
- Percentage-Based Results - Shows fake vs. real probability (0-100%)
- Dual Input Modes - Paste text directly or analyze via URL
- Modern UI - Clean, responsive interface with animated results
- Free to Use - Completely free with Hugging Face's inference API
